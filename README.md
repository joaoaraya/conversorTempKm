# Conversor
De temperaturas em:
* Kelvin
* Celsius
* Fahrenheit

De medidas em:
* Centimetros
* Metros
* Kilometros

> Para abrir o programa basta navegar até: `Conversor/bin/debug/Conversor.exe`
